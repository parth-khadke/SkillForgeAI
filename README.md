# SkillForge AI

SkillForge AI is an AI-powered learning platform concept designed to help learners escape "tutorial hell" through structured, validated progress.

Instead of consuming random content, users define a goal and receive a personalized roadmap. Each learning module is unlocked progressively after passing validation, so progress is based on demonstrated understanding rather than passive completion.

## Live Demo

The website is live on GitHub Pages:

https://parth-khadke.github.io/SkillForgeAI/

## What This Project Does

- Generates goal-based, personalized learning roadmaps
- Breaks learning into structured weekly modules
- Uses validation checkpoints (quizzes/challenges) to unlock next steps
- Encourages mastery-first progression over content bingeing

## Product Vision Highlight

SkillForge AI focuses on validated, adaptive learning:

- Personalized paths based on user goals, level, and time availability
- AI-generated assessments after each module
- Progressive locking to prevent skipping fundamentals
- Feedback loops for retries and improvement
